# Firefly Controller

Native Android App for Firefly AI Control

## Features

- Native Android UI (Kotlin + Jetpack Compose)
- Real-time status monitoring
- Task history
- Command sending
- Encrypted token storage

## Architecture

- MVVM with ViewModel
- Retrofit for HTTP
- EncryptedSharedPreferences for secure storage

## API Server

Requires firefly_api.py running on port 5000
